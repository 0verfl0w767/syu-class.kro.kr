unofficial_syu_api_docs

본 문서는 비공식 삼육대학교 강의계획서 API 문서 입니다.

학부(과)에 정상적인 데이터 값이 없는 경우 통폐합 또는 폐지된 과입니다.

학부(과)별 강의계획서 불러오기 사용법 안내

[GET] https://syu-class.kro.kr/api/undergraduate/v1/{학부(과) 식별 번호}

현재 사용 불가.

학부(과) 전체 목록 불러오기 사용법 안내

[GET] https://syu-class.kro.kr/api/college/v1/all

EX REQUEST

[GET] https://syu-class.kro.kr/api/college/v1/all

EX RESPONSE

{ "api": [ { "단과대학": "신학대학", "학부(과)": "신학과", "식별번호": "1", }, ... ] }
